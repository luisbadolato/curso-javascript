2. Operaciones B√°sicas con Arrays

2.1. Agregar y eliminar elementos

push() - Agregar al final

const frutas = ['manzana', 'banana'];
frutas.push('naranja'); // Agrega 'naranja' al final
console.log(frutas); // ['manzana', 'banana', 'naranja']

// Puede agregar m√∫ltiples elementos
frutas.push('pera', 'uva');
console.log(frutas); // ['manzana', 'banana', 'naranja', 'pera', 'uva']

pop() - Eliminar del final

const frutas = ['manzana', 'banana', 'naranja'];
const ultimaFruta = frutas.pop(); // Elimina y retorna 'naranja'
console.log(ultimaFruta); // 'naranja'
console.log(frutas); // ['manzana', 'banana']

unshift() - Agregar al inicio

const numeros = [2, 3, 4];
numeros.unshift(1); // Agrega 1 al inicio
console.log(numeros); // [1, 2, 3, 4]

// Puede agregar m√∫ltiples elementos
numeros.unshift(-1, 0);
console.log(numeros); // [-1, 0, 1, 2, 3, 4]

shift() - Eliminar del inicio

const numeros = [1, 2, 3, 4];
const primerNumero = numeros.shift(); // Elimina y retorna 1
console.log(primerNumero); // 1
console.log(numeros); // [2, 3, 4]

2.3. Longitud de un array: length

const animales = ['perro', 'gato', 'conejo'];

// Obtener la longitud

// Usar length para acceder al √∫ltimo elemento

// Aumentar la longitud crea espacios vac√≠os
