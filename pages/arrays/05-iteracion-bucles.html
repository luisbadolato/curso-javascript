5.1. Uso de bucles para recorrer arrays

Bucle for tradicional:

    console.log(`√çndice ${i}: ${frutas[i]}`);

Bucle for...of (recomendado para arrays):

    console.log(color);

// verde

Bucle while:

let i = 0;

while (i < numeros.length) {
    console.log(numeros[i]);
    i++;
}

Bucle do...while:

const animales = ['perro', 'gato', 'conejo'];
let j = 0;

do {
    console.log(animales[j]);
    j++;
} while (j < animales.length);
